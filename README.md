## About This Repository

This repo documents my daily Python learning journey through Angela Yu's
100 Days of Code course on Udemy. Each day has its own folder containing
the project code and an output screenshot. Every project is written,
tested, and improved by me from scratch — not just copied.

The end goal isn't just to finish the course. It's to build a strong
Python foundation that I can take into Machine Learning and AI.

---

## What's Inside Each Day Folder

```
Day-XX/
├── project_name.py     # the main code
└── output.png          # screenshot of the output
```

## How to Run Any Project

```bash
# 1. Clone the repo
git clone https://github.com/samirkshaw/100_Days-100_python-Projects.git

# 2. Navigate to a day
cd Day-XX

# 3. Run the file
python main.py
```
